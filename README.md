# 7Hive

7Hive, Yeditepe Üniversitesi öğrencileri, akademisyenler ve girişimciler için geliştirilmiş bir sosyal ağ ve proje yönetim platformudur.

## Proje Hakkında

7Hive, kampüsteki topluluklar arası iletişimi güçlendirmek ve işbirliğini kolaylaştırmak amacıyla tasarlanmış bir mobil uygulamadır. Kullanıcılar portföylerini sergileyebilir, proje açabilir, iş ilanları paylaşabilir veya kişisel çalışmalarını portfolio olarak sunabilirler.

## Teknolojiler

- **Dil:** Kotlin
- **UI Framework:** Jetpack Compose (Material 3)
- **Mimari:** MVVM + Repository Pattern
- **Backend:** Firebase (Authentication, Firestore, Storage)
- **Min SDK:** 24 (Android 7.0)
